# Introduction

The purpose of SDC Context Builder is to encapsulate the complexity of interaction with SDC component. It allows to encapsulate the complexity of reaching a component as well as implement the mapping and transformation componentâ€™s related data to common model consumed by audit process.

# Compiling POMBA Audit Common Model

POMBA SDC Context Builder can be compiled easily using maven command: `mvn clean install`
The result is JAR file under "target" folder
